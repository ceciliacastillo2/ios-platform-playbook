# Decisions

Architecture Decision Records (ADRs) for the ClaraCard iOS app.

ADRs capture significant technical decisions — what was decided, why, and what alternatives were considered. They're written once and rarely change, so future engineers understand the context behind the current design.

## How to add an ADR

Create a new file named `NNN-short-title.md` where `NNN` is the next number in sequence. Use the template in [`/templates/`](../../templates/).
